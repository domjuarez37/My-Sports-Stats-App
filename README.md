# Sports Stats Database 
### Enter a popular player to test it out!

Welcome to my player stats database. To use this app, simply put in any player from the MLB, NFL, or NBA. You will then be given that player's career statistics. For MLB players this includes homeruns, RBIs, and stolen bases. For NFL offensive players this includes TDs, total yards, and receptions. For NFL defensive players this includes tackles, sacks, and INTs. For NBA players this includes points, rebounds, and assists.

A comparison of 2 players is currently in the process of being added.

The database is fairly new and will expand in size over time. I hope you enjoy this app!
